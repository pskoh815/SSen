## 접속

```powershell
# 서버 시작
.\run.ps1 ui_up

# 브라우저 접속
http://localhost:8000/dashboard
```

## 주요 기능

### 기간 선택
- 날짜 직접 입력 또는 `3개월` / `6개월` / `1년` / `전체` 단축 버튼
- 첫 로드 시 기본값: 최근 6개월 자동 조회

### 데이터 최신성 표시 (상단)
- **데이터 기준일**: ETL 최종 날짜 (max_date), 7일 이내면 초록색
- **마지막 업데이트**: ETL 완료 시각
- **버전**: dataset_version

### 레짐 타임라인
- 가로 막대: 각 주도 테마의 지속 구간
- 테마별 고유 색상, 막대 클릭 시 해당 테마로 필터링
- 툴팁: 테마명, 기간, 리더 종목명

### 트레이드 로그
- 컬럼 클릭으로 정렬
- 텍스트 검색: 종목명/코드/테마명
- 청산 사유 필터: regime_end / stop_loss / take_profit / open
- 수익/손실 필터
- 행 클릭 → 근거 패널 활성화

### 근거 패널 (Explainability)
트레이드 행을 클릭하면 5단계 근거 자동 생성:
1. 신호일(t): 어떤 테마가 왜 선택됐는지
2. 종목 선정 이유 (테마 내 거래대금 최상위)
3. 진입 체결 (t+1 종가)
4. 청산 체결 및 사유
5. 수익률 결과

### 클라이언트 캐시
- 동일 기간 재조회 시 API 미호출 (Map 캐시)
- 새로고침(F5) 시 캐시 초기화

## 네트워크 오류 처리
- API 서버 미응답 시 상단에 오류 메시지 + 안내 표시
- 빈 결과: "조건에 맞는 데이터가 없습니다" 표시
